# Análisis de Datos — Cyclistic Bike Share

---

## Tecnologías Utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Proceso de Trabajo
### 1. Limpieza de Datos
- eliminación de valores nulos
- corrección de formatos
- tratamiento de inconsistencias
- validación de registros

### 2. Análisis Exploratorio (EDA)
- análisis temporal
- segmentación de usuarios
- duración promedio de viajes
- frecuencia de uso
- análisis semanal y mensual

### 3. Visualización de Datos
Se desarrollaron gráficos para:
- identificar tendencias
- detectar patrones de comportamiento
- comparar segmentos de usuarios

---

## Principales Insights
- Los usuarios casuales presentan mayor actividad durante fines de semana.
- Los miembros utilizan el servicio con mayor regularidad en días laborales.
- Existen patrones temporales que podrían utilizarse para campañas de conversión.
- Determinados períodos presentan picos de demanda que pueden afectar la operación.

---

## Conclusiones
El análisis permitió comprender diferencias de comportamiento entre usuarios y detectar oportunidades de mejora tanto para marketing como para operaciones.

Este proyecto fortalece habilidades de:
- análisis de datos
- visualización
- limpieza de datos
- interpretación de resultados
- storytelling analítico
